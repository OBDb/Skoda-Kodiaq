# Škoda Kodiaq

This repository contains signal set configurations for the Škoda Kodiaq, organized by model year and version. The files are structured to allow for easy differentiation between model generations and other vehicle parameters, ensuring accurate signal mapping for each version of the Škoda Kodiaq.

The Škoda Kodiaq is a mid-size crossover SUV manufactured by Czech automaker Škoda Auto since 2016. Named after the Kodiak bear from Alaska, it represents Škoda's entry into the large SUV segment, positioned above the Kamiq and Karoq models.
